
# Singing Trainer

Project học AI audio để test vocal range và transpose bài hát.

## Chức năng chính
- Upload voice sample
- Detect vocal range
- Tách vocal (Demucs)
- Pitch analysis
- Transpose bài hát

## Tech
- Backend: FastAPI, Python
- Frontend: React (Vite)

## Chạy project

### Backend
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
````

### Frontend

```bash
cd frontend
npm install
npm run dev
```
